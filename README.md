
A modern, responsive login and registration portal built with HTML, CSS, and JavaScript. Features smooth animations, password strength indicator, and form validation.

## Features

- User-friendly login and registration forms
- Password strength meter
- Responsive design for mobile and desktop
- Smooth transitions and animations
- Form validation and error handling

## How to Run

1. Clone or download the project files.
2. Open `index.html` in your web browser.
3. Interact with the login and register forms.

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)
- Font Awesome for icons
- Google Fonts for typography

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge
